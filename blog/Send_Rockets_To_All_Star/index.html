Send Harden and Howard to the All-Star Game using Automation

by Fabian Buentello
12/31/2015

Introduction

I wish I would've done this sooner, but better late than never! OK, so this is going to be a really short tutorial on how to automate a text message using Automator on your Mac. We'll be sending out text messages once a day to submit our vote for the 2016 NBA All-Star Game. I will be voting for the best one-two punch in the game right now, James Harden and Dwight Howard. We're struggling a bit this year, but it's all part of the show baby ;).

What is needed.

  • A Mac running Mountain Lion(Version 10.8) or later.
  • Messages (comes with Mac by default)
  • iCal (comes with Mac by default)
  • Automator (comes with Mac by default)

If you don't know how to check your OS version, check out this link: What's my Mac OS?. Same for Messages, if you don't know how to set it up, follow this tutorial: Setup iMessage on Mac. Most importantly, make sure you have Text Message Forwarding setup.

I repeat, make sure you have Text Message Forwarding setup.

Step 1. Automator

Pretty Straight forward, hit CMD+Space to open up the spotlight and type in "Automator".

Select New Document.

Select Calendar Alarm.

Select Utilities then drag Launch Application to the right section of the window. And Select Messages from the dropdown inside of the Launch Application block.

Step 2. Run AppleScript

Next, drag Run AppleScript below the Launch Application block, and paste the following code.

#Delay to give Messages enough time to open delay 5 tell application "Messages" #The number 696-22 is the All-Star voting number send "Harden" to buddy "696-22" of service "SMS" send "Howard" to buddy "696-22" of service "SMS" end tell

Since applications take time to open, we're going to delay for 5 seconds a bit before sending out the text messages. I wanted to keep this script as simple as possible since most readers won't be programmers. Of course, you can change the text to vote in other players. Just know that if more than one player has the same last name you will have to change the script to handle that.

Now let's hit the big Play button in the Run AppleScript block and you should see Messages open up and send a text message to 696-22.

By the way, I saved the number as AllStar Vote, so that's why you're not seeing the number at the top of my message window.

Alright, if all is good, let's go back to Automator and hit CMD+S to save our file and name it whatever you want. After, it should open up iCal with a new event on your calendar.

Step 3. Calendar

Right now it is December 31st at 11:35 AM, so to make sure the automation worked, I moved the event to December 30th and set the time to 11:40 AM. Next, I set repeat to Every Day and set end repeat to 01/19/2016 since that's when voting ends.

Conclusion

Well I hope you found this post interesting and useful, this type of automation could be used for a number of things. If you're not already automating, this is a great way to start.
